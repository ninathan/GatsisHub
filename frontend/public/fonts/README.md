# Three.js Fonts

For 3D text rendering, you need to place Three.js font files here.

## Download Font:

1. Go to: https://github.com/mrdoob/three.js/tree/dev/examples/fonts
2. Download `helvetiker_regular.typeface.json`
3. Place it in this `public/fonts/` directory

Or use this direct link:
https://raw.githubusercontent.com/mrdoob/three.js/dev/examples/fonts/helvetiker_regular.typeface.json

## Alternative Fonts:

- `helvetiker_bold.typeface.json`
- `optimer_regular.typeface.json`
- `gentilis_regular.typeface.json`

The font file is used by Text3D component in HangerScene.jsx
